    **src/http/session.ts:**
    Export `ArcSession` interface `{ cartToken: string | null; nonce: string | null }`.
    Export `SessionStore` interface `{ get(): ArcSession; set(patch: Partial<ArcSession>): void }`.
    Export `createInMemorySessionStore(): SessionStore` — simple closure over a mutable `ArcSession` state object. No external imports.

    **src/http/errors.ts:**
    Export `ArcError` as a discriminated union:
    ```
    type ArcError =
      | { type: 'api'; status: number; code: string; message: string; data?: unknown }
      | { type: 'network'; message: string; cause?: unknown }
      | { type: 'parse'; message: string; cause?: unknown }
    ```
    Export `normaliseError(status: number, body: unknown): ArcError` — if body has `code` and `message` string fields, return `{ type: 'api', ... }`; otherwise return `{ type: 'api', status, code: 'unknown', message: String(body) }`.
    Export `class ArcRequestError extends Error` that wraps `ArcError` for throw semantics: stores `.arcError: ArcError` on the instance.

    **src/http/client.ts:**
    Export `WooClientConfig` interface:
    ```
    { baseUrl: string; session?: SessionStore; wpBaseUrl?: string }
    ```
    Export `WooClient` class:
    - Constructor: stores `baseUrl` (strips trailing slash, appends `/wp-json/wc/store/v1`), stores `wpBaseUrl` (defaults to `baseUrl`), stores `session` (defaults to `createInMemorySessionStore()`).
    - `async request<T>(method: string, path: string, options?: { body?: unknown }): Promise<T>`:
      1. Read session — get `cartToken` and `nonce`.
      2. Build headers: always `Content-Type: application/json`. If `cartToken` is non-null, add `Cart-Token: {cartToken}`. ONLY if `cartToken` is null AND `nonce` is non-null, add `X-WC-Store-API-Nonce: {nonce}` (mutually exclusive — never co-send).
      3. Call `fetch(url, { method, headers, body: JSON.stringify(options?.body) })`.
      4. Capture returned `Cart-Token` response header — if non-null and different from current, call `session.set({ cartToken })`.
      5. If `!response.ok`, delegate to `private handleError<T>()`.
      6. Return `response.json() as Promise<T>`.
    - `private async handleError<T>(response, method, path, options, isRetry = false): Promise<T>`:
      1. Parse body with `response.json().catch(() => ({}))`.
      2. If `status === 401 && body.code === 'woocommerce_rest_cookie_invalid_nonce' && !isRetry`: call `await this.refreshNonce()`, store result via `session.set({ nonce })`, call `this.request<T>(method, path, options)` marking internal retry flag (pass a private sentinel to prevent second retry).
      3. Otherwise: `throw new ArcRequestError(normaliseError(response.status, body))`.
    - `private async refreshNonce(): Promise<string>`: fetch `{wpBaseUrl}/wp-json/arc/v1/nonce`, parse JSON, return `data.nonce as string`. Throws `ArcRequestError` with `type: 'network'` if fetch fails.

    Do NOT import from `next/*`. No `server-only` in this file (WooClient runs in browser and server). Keep each file under 200 lines — split if needed.

    **Package dependencies:** Add to Arc/Core/package.json: `"zod": "catalog:"` under dependencies. The `graphql-request`, `graphql` deps will be added in Task 2.
