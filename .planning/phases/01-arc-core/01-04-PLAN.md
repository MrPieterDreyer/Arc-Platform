<!-- WPGraphQL product types (from RESEARCH.md) -->
Union types: SimpleProduct | VariableProduct | ExternalProduct | GroupProduct
Fragment pitfall: Fragments on SimpleProduct cannot spread on VariableProduct.
Always use inline fragments: ... on SimpleProduct { } ... on VariableProduct { }
